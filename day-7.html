CallStack:

-It is a stack data structure which stores all the execution contexts created during the code execution in a LIFO(Last In First Out) manner.

Memory Heap:

-It is a large unstructured region of memory which is used for dynamic memory allocation where variables are allocated memory at runtime.

Execution Context Stack:

-It is the combination of Call Stack and Memory Heap.
